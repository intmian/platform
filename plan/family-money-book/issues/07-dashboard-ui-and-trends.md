# Issue 07: 数据看板 UI 和趋势

## 阶段

第二部分：数据看板

## 目标

为管理员和授权看板用户提供只读家庭账本看板，展示当前关键指标、资产负债结构、历史趋势和大事记。

## 范围

1. 前端路由：
   - `/money/:bookId/dashboard`
2. 看板展示：
   - 当前净资产
   - 当前现金
   - 当前负债
   - 总资产
   - 净资产变化
   - 投资盈利
   - 净资产负债率
   - 资产负债率
   - 资产结构
   - 负债结构
   - 历史趋势
   - 大事记
3. 管理员和普通授权用户共用同一个只读看板入口。
4. 普通授权用户不显示配置、对账、历史管理入口。

## 待确认

1. 普通用户看板是否显示项目明细金额。
2. 默认展示汇总看板，还是明细看板。

如果未确认，第一版按汇总看板处理：

1. 显示总数和趋势。
2. 不显示单个账户、投资、债务、债权明细。

## 非目标

1. 不做对账编辑。
2. 不做 Excel 导入。
3. 不显示平账建议。

## 验收标准

1. 管理员可以从账本进入看板。
2. 授权普通用户可以直接打开看板。
3. 未授权用户不能查看看板数据。
4. 当前指标与最近确认批次一致。
5. 历史趋势来自确认批次，不包含未确认草稿。
6. 大事记按批次日期展示。
7. 前端完成管理员看板、授权用户看板、未授权拦截的交互验证。
